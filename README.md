# RayTracingNextWeekCS
Ray Tracing the Next Week implemented in c#

In program.cs, modify outputFile to set an output file name, and texturePath to set the path to the texture file.

To run, navigate into
 src\RayTracerNextWeekCS
 
and invoke:

dotnet run --property:Configuration=Release 9

The last parameter is the scene parameter
